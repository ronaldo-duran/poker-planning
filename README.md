# poker-planning
A Planning Poker web application built with Laravel, Vue, and WebSockets, created using AI-assisted development and autonomous coding agents.

## Agents and Skills

This project uses a modular structure with agents and skills to manage different aspects of development:

- **Agents**: High-level responsibilities for backend, frontend, database, infrastructure, real-time communication, and UI.
- **Skills**: Detailed instructions for specific tasks related to each agent.

Refer to the `project-context.md` file for more details.
